CheckPoint Game Hub is a game tracker where users can create a profile and browse games, save favorites, mark games as played or in progress, give a star rating and review to games, and filter games by genre and other game related filters. 

Tech stack:
- HTML
- CSS (Tailwind)
- JavaScript
- Firebase for authentication and storage

Features:
1. User authentication
2. Cloud database storage with firebase
3. Browse/search games
4. Favorites system
5. Rating and reviews
6. Filter and sorting for searches
7. dark mode toggle
8. Editable user profile
9. Let users set a visible profile status
